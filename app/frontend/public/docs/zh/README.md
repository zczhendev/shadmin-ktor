### 技术栈速览

- **前端**: React 19 + TypeScript + Vite + Tailwind CSS v4 + shadcn/ui + TanStack Router/Query
- **后端**: Ktor 3.0 + Kotlin 2.1 + Exposed ORM
- **数据库**: H2 (开发) / PostgreSQL / MySQL
- **模块化**: Google AutoService SPI
